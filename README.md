# Soar React Components

A modern React design system component library built with Tailwind CSS, Radix UI, and Motion.

## Installation

```bash
npm install @soar-design/soar-react-components
```

## Usage

```tsx
import { Button, Card, Input } from '@soar-design/soar-react-components'
import '@soar-design/soar-react-components/globals.css'

function App() {
  return (
    <Card>
      <Button>Click me</Button>
      <Input placeholder="Enter text" />
    </Card>
  )
}
```

## Development

```bash
# Install dependencies
npm install

# Build the library
npm run build

# Run development mode
npm run dev
```
